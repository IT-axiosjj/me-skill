# me-skill 是做什么的

> 一个用于构建个人分身资料的 Claude Code skill。

[返回首页](./README.md) · [安装与使用](./INSTALL_AND_USAGE.md) · [Skill 入口](./SKILL.md)

`me-skill` 的目标不是只模仿语气，而是把你的材料拆成更稳定的结构：

- 你是谁
- 你怎么说话
- 你重视什么
- 你的边界是什么
- 哪些表达不像你

---

## 它解决什么问题

很多“数字分身”方案容易只停留在表层语气模仿，结果常见问题是：

- 风格像一点，但不稳定
- 事实、价值观、边界混在一起
- 不知道什么是像你，什么明显不像你

`me-skill` 想做的是把这些层分开。

---

## 它输出什么

生成后会得到：

```text
self.md
persona.md
profile.md
records.json
```

| 文件 | 作用 |
| --- | --- |
| `self.md` | 事实、偏好、价值观、边界 |
| `persona.md` | 表达风格、情绪态度、决策倾向、反例 |
| `profile.md` | 最终摘要与模拟规则 |
| `records.json` | 累计输入材料 |

---

## 它适合谁

- 想做自己的 Claude Code 分身 skill 的人
- 想整理长期表达风格的人
- 想把零散文本材料变成结构化档案的人
- 想继续扩展成更完整 skill 产品的人

---

## 它怎么工作

当前主要吃三类材料：

| 类型 | 说明 |
| --- | --- |
| notes | 自述、日记、偏好、原则、个人说明 |
| chat | 真实对话样本，用来提取表达方式、节奏、互动风格 |
| correction | “不像我”“我不会这样说”“更像我的表达”这类纠正材料 |

然后生成：

- `self.md`
- `persona.md`
- `profile.md`
- `records.json`
