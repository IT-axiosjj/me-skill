# me-skill 是做什么的

## 一句话介绍

`me-skill` 是一个把“你的材料”整理成“你的个人分身档案”的工具。

它不是简单模仿语气，而是尽量把下面几件事拆开：

- 你有哪些稳定偏好
- 你重视什么
- 你说话是什么风格
- 你的情绪表达是什么样
- 你通常怎么做决定
- 哪些表达明显不像你

## 它解决什么问题

很多“数字分身”类工具会把所有东西混在一起，只生成一个很像人设的描述，但不够稳定。

`me-skill` 想解决的是：

1. **把事实和风格分开**
   - `self.md` 负责“你是谁”
   - `persona.md` 负责“你怎么说话”

2. **把边界和反例留下来**
   - 很多时候，知道“你不像什么”比知道“你像什么”更重要

3. **支持持续修正**
   - 你可以不断补材料
   - 你可以说“这不像我”
   - 系统会重新生成并保留旧版本

4. **让个人材料变成结构化资产**
   - 不是一次性聊天记录
   - 而是可复用、可迭代、可展示的个人档案

## 适合谁用

这个项目适合：

- 想做自己的 AI 分身的人
- 想沉淀表达风格和思维方式的人
- 想整理长期个人资料的人
- 想把零散文本、聊天、纠正语句变成结构化档案的人

## 它的工作方式

`me-skill` 目前主要吃三类材料：

### 1. notes
自述、日记、偏好、原则、个人说明。

### 2. chat
真实对话样本，用来提取表达方式、节奏、互动风格。

### 3. correction
“不像我”“我不会这样说”“更像我的表达”这类纠正材料。

然后输出：

- `self.md`
- `persona.md`
- `profile.md`
- `records.json`

## 这个 skill 的核心价值

### 1. 让“我是谁”更清楚
很多人知道自己大概是什么风格，但很难系统整理出来。

### 2. 让“我怎么说话”更可复用
它可以把你真实表达中的稳定部分沉淀下来。

### 3. 让“边界”变得明确
AI 最容易模仿错的地方，往往是情绪强度、措辞方式和价值判断。
`me-skill` 会尽量把这些边界提前写清楚。

### 4. 让分身不是一次性产物
你可以持续追加材料，不断更新，而不是生成一次就作废。

## 当前版本定位

当前版本不是一个全自动、强理解的大模型人格系统。

它更像：

- 一个轻量的个人分身构建器
- 一个可跑通的最小实用版本
- 一个适合继续扩展的基础仓库

## 你可以怎么展示它

在 GitHub 上，这个项目可以被理解成：

- 一个 Claude Code skill 项目
- 一个个人数字分身整理工具
- 一个从文本材料生成 me-profile 的工作流模板

## 未来可以扩展什么

后续可以继续扩展成：

- 直接支持微信 / QQ 导出
- 更强的规则抽取器
- 基于模型的深度归纳
- 多 persona / 多版本 me 管理
- 真正的 `/create-me`、`/update-me`、`/list-me` 命令体系
