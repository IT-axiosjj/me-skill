# install

如果你想把这个项目作为 Claude Code 本地 skill 使用，可以复制到以下任一位置：

## 项目级安装

放到：

```text
.claude/skills/me-skill/
```

## 全局安装

放到：

```text
~/.claude/skills/me-skill/
```

## 至少需要的文件

- `SKILL.md`
- `prompts/`
- `selves/`（可为空）

## 当前仓库内的便捷镜像目录

仓库中已提供：

```text
.claude/skills/me-skill/
```

你可以直接把这里的内容复制到目标位置。
